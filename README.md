# BEQUEATH

Game to try out making a magic system  
Making the engine in Rust, and the whole game in Rust too 

## Libraries used:
 - Rust standard libraries
 - `tiny-skia`
 - `softbuffer`
 - `winit`
 - `image`
 - `rodio`

## Ideas
 - [ ] Freeform magic circle creation
 - [ ] Earth element
 - [ ] Fire element
 - [ ] Lightning element
 - [ ] Water element
 - [ ] Roguelike
 - [ ] Procedurally generated spire for each attempt
